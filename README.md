# Eduardo Garcia's web portfolio 2.0

The main ideia of this project is not to create just an ordinary website, but to create a memorable experience for the user with beautiful layouts and out-of-the-box animations. Also, in the future, i will try to earn SOTD premiation from [](https://www.awwwards.com/)

## Technologies i've used

- [Nuxt](https://nuxt.com/)
- [Nuxt i18n plugin](https://i18n.nuxtjs.org/)
- [GSAP](https://gsap.com/)
- [Lenis Scroll](https://lenis.studiofreight.com/)

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
